# Portfolio Dump

This repository contains different portfolio website experiments, design iterations, animation concepts, and UI layouts that I tried while exploring modern personal portfolio designs.

The goal of this repo is not to maintain one final polished website, but to collect multiple versions, compare ideas, and reuse the best sections later.

## Contents

- Portfolio layout experiments
- Hero section designs
- Scroll animation concepts
- Interactive UI ideas
- Responsive design tests
- Unused or rejected portfolio versions
